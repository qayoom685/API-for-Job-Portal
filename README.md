# API-for-Job-Portal


 API for Job Portal

A RESTful API designed for managing job listings and intern applications, built to support Internee.pk’s recruitment system.
This backend service allows companies to post job opportunities and interns to apply seamlessly through structured API endpoints.

 Objective

Build a scalable backend API that handles job postings, manages applications, and interacts efficiently with clients (web/mobile).

 Features
 Job Listings

Create, update, delete job posts

Fetch all jobs or a single job

Store job details like title, description, company, salary, etc.

 Intern Applications

Apply for a job via API

Store applicant details securely

 Additional Capabilities

Proper error handling

Clean and modular architecture

Easily expandable for future features

Tech Stack

Technology	Purpose
Node.js	Backend server
Express.js	Routing & middleware
MongoDB + Mongoose	Database & ORM
Postman	API testing
